Iris Flower Classification App 

A machine learning web application built with Random Forest Classifier and Streamlit to predict Iris flower species based on flower measurements.

📌 Project Overview

This project uses the famous Iris dataset to classify flowers into one of the following species:

Setosa
Versicolor
Virginica

The application provides an interactive frontend built with Streamlit, allowing users to input flower measurements and get real-time predictions.

🚀 Features
Machine Learning model using Random Forest Classifier
Interactive Streamlit web interface
Real-time predictions
Simple and user-friendly UI
Fast and lightweight application
